## 项目简介

Ollama 英特尔优化版是一个专为英特尔硬件优化的桌面应用解决方案，用于在 Windows 和 Linux 操作系统上运行和管理 Ollama 模型的 GUI 工具。

该版本特别针对英特尔处理器、Intel Arc GPU 和相关技术栈进行了深度优化，提供最佳性能体验。
